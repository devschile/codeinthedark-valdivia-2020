Sobre el evento

Code in the Dark Valdivia

2020

Code in the Dark es una competencia originalmente creada en Europa por un pequeño grupo de desarrolladores, donde cada concursante compite para implementar el diseño de una pieza web solo con una captura de pantalla. En esta no se permiten vistas previas de los resultados durante la implementación, y no se pueden utilizar herramientas de medición.

Competencia amistosa

Los concursantes compiten en una ronda con su computador, el cual contiene el editor "especial" de Code in the Dark. Es duro y divertido, además la multitud agrega una nueva dimensión.

Ambiente de fiesta

No es una simple competición, es una fiesta con DJ, luces, efectos, música trance y todo lo que un hacker de película haya soñado en un solo ambiente.

Todos son bienvenidos

No tienes que competir para divertirte en Code in the Dark. Cada miembro de la audiencia puede apoyar a su favorito y ser parte de esta gran fiesta.

Un ganador
En esta edición de Code in the Dark Valdivia habrá desafíos del tipo frontend y backend.
Conocer más

Para el desafío frontend cada participante recibe una captura de pantalla y una serie de instrucciones. Construyen el diseño proporcionado usando sólo HTML y CSS puro.

En el caso del desafío backend los participantes deben solucionar algún problema acotado con un lenguaje a elección. Por ejemplo manipular una colección de datos, realizar búsquedas, mapear, etc.

Se proporcionarán todos los assets e instrucciones necesarias. No se pueden usar frameworks, librerías o recursos externos. Las vistas previas de los resultados no están permitidas durante la competencia. Una vez que se agota el tiempo de 15 minutos cada participante presenta su resultado a la audiencia, la cual es la encargada de decidir el ganador.
